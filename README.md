Game Tap — Android App

Game Tap — это кроссплатформенное мобильное игровое приложение, разработанное с использованием React Native и Expo.
Игра ориентирована на развитие реакции и внимательности пользователя за счёт взаимодействия с элементами интерфейса посредством нажатий на экран.
Используемые технологии

React Native — кроссплатформенная мобильная разработка

Expo — упрощение сборки, тестирования и запуска приложения

Expo Router — файловая навигация

TypeScript — строгая типизация

expo-haptics — хаптик-фидбек

expo-image-picker — камера и галерея

Android SDK — сборка и запуск под Android
