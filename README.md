Game Tap — Android App

Game Tap — это кроссплатформенное мобильное игровое приложение, разработанное с использованием React Native и Expo.
Игра ориентирована на развитие реакции и внимательности пользователя за счёт взаимодействия с элементами интерфейса посредством нажатий на экран.
Используемые технологии

React Native — кроссплатформенная мобильная разработка

Expo — упрощение сборки, тестирования и запуска приложения

Expo Router — файловая навигация

TypeScript — строгая типизация

expo-haptics — хаптик-фидбек

expo-image-picker — камера и галерея

Android SDK — сборка и запуск под Android
Установка зависимостей
npm install

Запуск приложения в режиме разработки
npx expo start

Локальная сборка (Debug)
npx expo run:android
Политика конфиденциальности

Приложение не собирает и не передаёт персональные данные пользователей.
Доступ к камере и галерее используется только по запросу пользователя и исключительно для функциональности приложения.

Назначение проекта

Проект разработан в рамках учебной дисциплины по мобильной разработке и используется для:

изучения React Native и Expo;

реализации платформо-специфичной функциональности;

подготовки приложения к возможной публикации в магазине приложений.
